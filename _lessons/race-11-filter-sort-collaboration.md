# RACE 11 - Web App: Filter & Sort Data + Collaboration - 13-05-2026

## Formål

Vi skal arbejde videre med jeres web app og samtidig klæde jer godt på til at samarbejde og aflevere jeres React-projekt i Customer Experience.

- Arbejde med filtrering og sortering af data i jeres React-app.
- Oprette et Git-repository og forbinde det til GitHub.
- Bruge source control i VS Code til at stage, committe og pushe ændringer.
- Samarbejde via branches, pull requests og merge.
- Deploye jeres React + Vite app til GitHub Pages.

## Forberedelse

- Skriv til RACE på Teams eller mail, hvis der er noget, du ønsker at få samlet op på eller afklaret.
- Noter dine spørgsmål, særligt til Supabase, CRUD, REST, HTTP og React.
- Genopfrisk sektionen [Filtrering & Sortering med Supabase REST](../README.md#7-filtrering--sortering-med-supabase-rest).
