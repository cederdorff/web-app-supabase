# RACE 11 - Web App: Filter & Sort Data + Collaboration - 13-05-2026

## Formål

Vi skal arbejde med at udvide jeres app med filtrering og sortering via Supabase REST query parameters og samtidig forberede os på at arbejde i grupper om en fælles web app. Målet er både teknisk forståelse og en god overgang til det afsluttende projekt.

## Forberedelse

- Du skal læse README-opgaven om filtrering og sortering.
- Du skal have dit projekt klar, så du kan teste filtre og sortering direkte i appen.
- Du skal genopfriske query parameters og state i React.
- Du skal tænke over, hvordan jeres gruppe kan fordele arbejdet i det afsluttende projekt.
- Du skal have GitHub klar, hvis I skal samarbejde i branches.

## Agenda

1. Vi repeterer kort CRUD og dataflowet mellem React og Supabase.
2. Vi introducerer filtrering og sortering med query parameters.
3. Vi tester filtrering og sortering manuelt mod Supabase.
4. Vi implementerer filtrering i React med state-baserede inputværdier.
5. Vi implementerer sortering og eventuelt limit i React.
6. Vi taler om brugeroplevelse: hvornår man skal hente data igen, og hvordan man viser aktive filtre.
7. Vi introducerer det afsluttende gruppeprojekt.
8. Vi planlægger samarbejdet i grupper med roller, opgaver og fælles aftaler.
9. Vi arbejder med gruppedannelse eller første projektplanlægning.

## Materialer

- README-opgaven i dette repository.
- Eksempler på Supabase queries med filter og sortering.
- React-komponenter med state til filter og sortering.
- Computer med adgang til projektet.
- Projektbeskrivelse til det afsluttende gruppeprojekt.
- Evt. skabelon til arbejdsfordeling i grupper.
